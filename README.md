## Most Useful VSCode Shortcuts Cheat Sheet

### General

#### 1. Show command pallete 
    Ctrl + Shift + P  or  F1

* Quick open, Go to file
    Ctrl + P 

* New window / instance
    Ctrl + Shift + N    

* Close window / instance
    Ctrl + Shift + W

* User settings 
    Ctrl + ,
    
* Keyboard 
    Ctrl + K Ctrl + S
